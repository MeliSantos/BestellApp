# Bestell App

Dies ist eine Web-basierte Bestell-App, die es Benutzern ermöglicht, Gerichte auszuwählen, in den Warenkorb zu legen und Bestellungen abzuschließen.

## Funktionen

- Anzeige von Hauptgerichten und Beilagen
- Interaktive Warenkorb-Funktion:
  - Hinzufügen von Gerichten
  - Mengenanpassung
  - Löschen von Artikeln
- Anzeige von Zwischensumme, Lieferkosten und Gesamtpreis
- Bestellbestätigung mit Overlay-Nachricht
- Responsive Navigation mit Burger-Menü

## Technologie-Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Strukturierte Daten für Gerichte in JS-Arrays

## Verwendung

Öffne index.html im Browser.

Wähle Gerichte aus den Kategorien Hauptgerichte und Beilagen.

Klicke auf das Plus-Symbol, um Gerichte zum Warenkorb hinzuzufügen.

Passe die Menge im Warenkorb an oder lösche Artikel.

Klicke auf "Bestellen", um die Bestellung abzuschließen. Ein Overlay zeigt die erfolgreiche Bestellung an.

## Anpassung

Gerichte können im myDishes und mySideDishes Array in script.js bearbeitet oder erweitert werden.

Preise, Beschreibungen und Bilder können einfach angepasst werden.
